# Nginx 反向代理配置

先说解决办法，检查反向代理服务器是否有配置：

```bash
proxy_set_header Host $host:$server_port;
proxy_set_header X-Forwarded-Proto $scheme;
```

如果没有或和**上面不一致**，请修改反向代理服务器配置文件，添加或修改上述两行。
