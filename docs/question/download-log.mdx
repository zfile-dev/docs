# 如何查看/下载日志？

## 后台下载

如能访问后台，可在登录后台下载日志。

<img className={'sm:w-2/3'} src='/img/2022/11/05/log.png'/>

## 服务器磁盘

日志文件在服务器磁盘中。非 Docker 部署默认位于 `~/.zfile-v4/logs/`；Docker 部署按容器创建时映射到 `/root/.zfile-v4/logs` 的宿主机目录查找。

通过一键脚本安装时，也可以在脚本目录执行：

```bash
./install.sh logs -n 200
./install.sh logs -f
```

手工 Docker 部署可执行 `docker logs --tail 200 zfile-pro` 查看容器输出。若是从旧开源版升级且沿用了原容器名，请将命令中的 `zfile-pro` 改为 `zfile`。
