# 上传失败怎么办？

## 上传超过 50M 的文件失败

上传时提示 `413 Request Entity Too Large`，这是因为上传的文件超过了服务器的限制，需要修改服务器的配置。

### 宝塔

宝塔 `nginx` 默认只支持上传最大 `50MB` 的文件，可去以下页面进行设置:

### Nginx Proxy Manager

编辑你的网站，在下图位置填写：

```
client_max_body_size 0;
```

表示无上传大小限制，也可以配置为限制指定大小，如： `client_max_body_size 1024m;`
