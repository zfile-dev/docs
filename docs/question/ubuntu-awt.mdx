# Ubuntu / Debian 登录页验证码无法显示怎么办？

当前版本使用 SVG 验证码，不依赖 Java AWT 或系统字体，也不需要安装 JDK 8 字体组件。

如果验证码区域无法显示，请依次检查：

1. 强制刷新浏览器页面，或清除站点缓存后重试。
2. 打开浏览器开发者工具的 Network，确认验证码接口是否返回成功。
3. 检查反向代理是否错误缓存或拦截 `/api/` 请求。
4. 查看 ZFile 日志中的对应请求错误，参考[如何查看 / 下载日志](/question/download-log)。

如果使用的是 4.x 旧版本，请优先升级；旧版本才可能需要安装 `fontconfig` 或处理 Java AWT 环境。
