# 配置文件路径（必看）

:::caution
注意，所有配置修改后都需要重启 ZFile 才能生效。
:::

如果你需要修改配置文件，那么你需要知道配置文件的路径，下面是各种部署方式下的配置文件路径：

<Tabs>
<TabItem value="os" label="开源版">

- [Linux](/install/guide#edition=os&platform=linux) — `~/zfile/application.properties`
- [Windows](/install/guide#edition=os&platform=windows) — 解压目录下的 `application.properties`
- [Docker](/install/guide#edition=os&platform=docker) — 宿主机 `/root/zfile/application.properties`（需映射配置文件）
- [宝塔Docker应用商店](/install/os-baota-new#config)
- [宝塔传统方式](/install/baota#config) — `解压路径/application.properties`



</TabItem>
<TabItem value="pro" label="捐赠版">

- [Linux](/install/guide#edition=pro&platform=linux) — `~/zfile-pro/application.properties`
- [Windows](/install/guide#edition=pro&platform=windows) — 解压目录下的 `application.properties`
- [Docker](/install/guide#edition=pro&platform=docker) — 宿主机 `/root/zfile-pro/application.properties`（需映射配置文件）
- [宝塔](/install/baota#config) — `解压路径/application.properties`

</TabItem>
</Tabs>
