# Debug 模式

:::warning 仅用于旧版本救援
5.0.0 及以上版本请优先使用本机 [`reset-admin` 救援命令](/question/reset-password)。Debug 模式会开放网页重置入口，不适合长期启用。
:::

旧版本忘记密码后，可修改配置文件开启 `debug` 模式，重启后在 ZFile 首页点击重置密码按钮，或访问 `http(s)://ip:port/api/site/reset-password`，将用户名重置为 `admin`、密码重置为 `123456`。

配置文件路径参考左侧的：**`参数配置 -> 配置文件路径（必看）`**，修改内容如下：

```properties showLineNumbers
# 默认为 false, 修改为 true 即可开启 debug 模式
zfile.debug=true
```

:::caution 提示

注意：重置完密码，请及时修改配置文件中的 `debug` 为 `false`，并重启服务，否则任何人访问你的网盘都可以重置密码！

:::
